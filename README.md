# قاموس هولندي - عربي

موقع قاموس تفاعلي للترجمة بين الهولندية والعربية مع إمكانية النطق والامتحانات.

## المميزات

- ✅ إضافة كلمات هولندية مع معانيها بالعربية
- ✅ إضافة معنى إضافي للكلمة
- ✅ الاستماع لنطق الكلمات والجمل بالهولندية
- ✅ إضافة جمل مع ترجمتها
- ✅ إنشاء امتحانات تفاعلية
- ✅ واجهة عربية بالكامل

## التثبيت

1. تثبيت المكتبات:
```bash
cd dutch-arabic-dictionary
npm install
```

2. إعداد قاعدة البيانات:
```bash
# انسخ ملف البيئة
cp .env.example .env

# عدّل DATABASE_URL في ملف .env

# تشغيل Prisma
npx prisma generate
npx prisma db push
```

3. تشغيل المشروع:
```bash
npm run dev
```

4. افتح المتصفح على: http://localhost:3000

## البنية

- `/words` - عرض وإضافة الكلمات
- `/sentences` - عرض وإضافة الجمل
- `/quiz` - إنشاء وحل الامتحانات

## قاعدة البيانات (Prisma)

النماذج:
- `Word` - الكلمات (هولندي، عربي، معنى إضافي)
- `Sentence` - الجمل المرتبطة بالكلمات
- `Quiz` - الامتحانات
- `QuizQuestion` - أسئلة الامتحانات

## التقنيات المستخدمة

- Next.js 14 (App Router)
- TypeScript
- Prisma ORM
- PostgreSQL
- Web Speech API (للنطق)
